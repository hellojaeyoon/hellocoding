## 계산기

중위표기법(`infix notation`) : 연산자를 피연산자의 가운데 표기

- A + B

후위표기법(`postfix notation`) : 연산자를 피연산자 뒤에 표기

- AB +

중위표기식을 후위표기식으로 변환해보자

- A * B - C / D

1: 수식의 각 연산자에 대해 우선순위에 따라 괄호를 사용하여 다시 표현한다.

`( ( A * B ) - ( C / D ) )`

2: 각 연산자를 그에 대응하는 오른쪽 괄호의 뒤로 이동시킨다.

`( ( A  B ) *  ( C  D ) / ) -`

3: 괄호를 제거한다

` A  B * C  D / -`
